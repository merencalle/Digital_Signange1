Sentinel — Content Admin (Standalone Mode)

Mission-Ready Digital Signage — Built for the Army

This is a lightweight, file-based editor for the standalone Tizen player's local JSON content. It has no backend, no login, and no audit trail — it just edits files on your machine. For multi-user, role-based, approval-gated content management, use the full Sentinel CMS web application instead.

Building / Zone Content

Load a building JSON file (e.g. content/buildings/bldg-a.json), edit it, and download the updated file back into place.

Emergency Alert (Global Override)

Fills in content/global/emergency-alerts.json. Every screen on every building polls this file every few seconds — setting it active overrides all screens immediately.
